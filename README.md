# The Simple Dice Game

### Description:
Develop Dice games using TypeScript, Next.js, and Material UI.

### AC:
- The user can enter a number (threshold) over or under which the dice roll must be.
- The user can select "Greater" or "Less" to indicate whether the result must be greater or less than the entered number.
- The user can click the "Play" button to see the result that meets the specified condition.
- The game result is a number from 1 to 100.
- If the result meets the condition (greater or less than the entered number, depending on the selected condition), a corresponding message or confirmation icon is displayed on the screen.
- The game result is stored in an array and the game history is displayed as a list.
- The maximum array size is 10 elements.

### Requirement:
- Using TypeScript for development.
- Using Next.js to create a web application.
- Using the Material UI library for interface styling.
